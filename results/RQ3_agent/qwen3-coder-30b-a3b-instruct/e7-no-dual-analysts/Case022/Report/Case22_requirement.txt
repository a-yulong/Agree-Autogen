The controller component shall define the boolean variables heating_request and cooling_request. The heating_request shall be true if and only if the user_request contains a heating command and does not contain a cooling command. Likewise, the cooling_request shall be true if and only if the user_request contains a cooling command and does not contain a heating command. Additionally, the controller shall ensure through assertions that: the heater_on output is true only if heating_request is true and the tempin input is less than or equal to 20 degrees; the cooler_on output is true only if cooling_request is true and the tempin input is greater than 0 degrees.
