The GCAS_Monitor component shall define vertical acceleration as 100.0, expected impact time as 15.0, and recovery time as 5.0. The GCAS_Monitor_impl implementation shall calculate the altitudeFromGround variable by invoking the Agree_Nodes::height function using the values of the altitude input port, the verticalVelocity input port, and a minimum height time of 0.5 seconds. Furthermore, the implementation shall guarantee that when the altitude input port value is less than or equal to 30.0 or the calculated altitudeFromGround value is less than or equal to 500.0, the gcasRequested output signal must be true.
