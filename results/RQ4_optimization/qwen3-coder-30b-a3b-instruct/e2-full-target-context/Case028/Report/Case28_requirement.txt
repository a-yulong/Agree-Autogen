The GCAS_singleton component shall maintain a real-valued recursive variable t with an initial value of 0.0 and a step size of 0.1. It shall define the constant minHeightAllowed as 200.0. At the implementation level, the component shall invoke the kinematicLinearization function from the gcas_nodes library, passing t as a parameter, to compute the height variable height, and shall set the output port timeToRecovery to the constant 3.7. The component shall guarantee that when the simulation time t reaches or exceeds the recovery time, the computed height value must always be greater than minHeightAllowed, and the current height value must be greater than the previous height value to indicate a pull-up trend.
