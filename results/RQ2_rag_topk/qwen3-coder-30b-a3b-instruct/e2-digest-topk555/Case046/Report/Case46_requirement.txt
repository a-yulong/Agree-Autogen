The top_level system must define an internal state variable mode. The environment assumes that the input signal Input is strictly less than 10. The system must guarantee that the state variable mode is always greater than or equal to 0, and that the output signal Output is strictly less than 50. In the implementation layer top_level.Impl, the input Input must be forwarded to the subsystem A_sub. The output of A_sub must be connected to both B_sub and C_sub as inputs. The output of B_sub must be connected to the second input of C_sub. The output of C_sub must drive the output Output of top_level. Additionally, the internal state variable mode of C_sub must be assigned to the state variable mode of top_level.
