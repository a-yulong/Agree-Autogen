Topic: agree_syntax_notes
AGREE contracts are written inside `annex agree {** ... **};` blocks. Component type annexes are the normal location for assumptions, guarantees, constants, and equation declarations. Component implementation annexes are used for implementation-level bindings such as assignments when they are valid in that scope.

Implementation-level bindings should be placed in implementation scopes after relevant subcomponents and connections. Subcomponent ports should not be used in a component type contract unless represented through valid variables or outputs.

If diagnostics involve an event port in a numeric or Boolean comparison, delete the clause unless the model provides an explicit data value for the event. Plain AADL event ports are not AGREE numeric values and should not be encoded as 0, 1, true, or false.
