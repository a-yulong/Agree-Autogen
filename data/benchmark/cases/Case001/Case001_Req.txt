When the TCAS_singleton component receives an ownAltitude_in value that equals the nearestIntruderAltitude value, if the ownLongitude in data port value is greater than the nearestIntruderLongitude value, then set the requiredVerticalVelocity out data port to 50; otherwise, set it to -50. When the ownAltitude_in value is less than the nearestIntruderAltitude value, set the requiredVerticalVelocity out data port to 10. When the ownAltitude_in value is greater than the nearestIntruderAltitude value, set the requiredVerticalVelocity out data port to -10. Additionally, define a boolean internal variable named ownAircraftGoesUpOnTie to store the result of the comparison between ownLongitude and nearestIntruderLongitude.
