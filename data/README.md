# Data directory

Place experiment cases under `data/Sources` or set `AGREE_SOURCE_ROOT` to another directory.

Expected layout:

```text
data/Sources/Case01_A/Case01_Base.txt
data/Sources/Case01_A/Case01_Req.txt
data/Sources/Case01_A/Case01/*.aadl
```

Generated reports should go under `results/` or the path configured by `AGREE_RESULT_ROOT`.
