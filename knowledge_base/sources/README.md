# Knowledge-Base Source Notes

The public sample KB is project-owned and intentionally small. It is designed to exercise retrieval and prompt-grounding behavior without redistributing license-unclear material.

To reconstruct a larger local KB, users may collect resources they are allowed to use locally, such as:

- Public AADL and AGREE tutorials or examples with compatible licenses.
- Project-owned notes derived from local validation experience.
- Locally prepared RequirementNL-LogicProp-CodeAGREE examples.
- Tool diagnostics observed from user-owned models.

Do not commit full standards, official manuals, course PDFs, private datasets, or third-party repository snapshots unless redistribution is explicitly permitted. For non-redistributable resources, keep them outside the repository and document source metadata only.
